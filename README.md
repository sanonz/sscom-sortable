### 排序 SSCOM 多字符串配置文件

例如把 `sscom51.ini` 文件的以下数据

```
N102=0,,1000
N2=A,1 string

N105=0,,1000
N5=A,4

N103=0,,1000
N3=A,3

N104=0,,1000
N4=A,2
```

重新挪动为


```
N102=0,,1000
N2=A,1 string

N104=0,,1000
N4=A,2

N103=0,,1000
N3=A,3

N105=0,,1000
N5=A,4
```

然后执行

```
npm run generate
```

生成文件在 `out/sscom51.ini`，把此文件复制到 `sscom5.13.1.exe` 同目录下，打开 `SSCOM` 工具查看排序后的多字符串列表
